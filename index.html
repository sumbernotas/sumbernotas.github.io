Artifacts Overview

This ePortfolio contains three artifacts that collectively demonstrate my technical versatility and growth as a software developer. Each artifact has been carefully selected and enhanced to showcase different aspects of my computer science capabilities:

Artifact One: Rescue Animal System - Software Design and Engineering Enhancement
The first artifact demonstrates my software engineering and design skills through the transformation of a command line appointment system (originally from CS-320) into a comprehensive full-stack web application. The original project used a simple command line interface with in-memory HashMap storage and basic CRUD operations. For Enhancement One, I completely redesigned the architecture using Spring Boot as the application framework and implemented the Model-View-Controller (MVC) pattern for proper separation of concerns. I developed a professional web interface using Thymeleaf templating engine and Bootstrap 5 for responsive design, creating an intuitive user experience for healthcare facilities to manage patients and appointments. The enhancement includes a multi-layered architecture with distinct Controller, Service, and Repository layers, each with clearly defined responsibilities. I implemented comprehensive server side validation using validation annotations, including regex pattern validation for names (letters, spaces, hyphens, apostrophes only), phone number format validation, email format validation, and future date validation for appointments. The application features automatic ID generation for patients (PAT1001, PAT1002...) and appointments (APT2001, APT2002...), ensuring unique identification throughout the system. This artifact showcases my ability to apply modern software design patterns, implement RESTful-style routing, create professional user interfaces, and transform legacy code into production-ready applications following industry best practices and SOLID principles.

Artifact Two: Binary Search Tree Implementation - Algorithms and Data Structures Enhancement
The second artifact focuses on algorithms and data structures through significant enhancements to a Java-based Rescue Animal management system originally developed in IT-145. I transformed the basic procedural program with ArrayList storage into a sophisticated system utilizing custom-implemented binary search trees for optimal performance. The key enhancements include replacing String types with appropriate datatypes (int for age, double for weight, LocalDate for dates), implementing an enum-based TrainingStatus system to ensure valid values and type safety, and expanding the system from supporting two animal types to four (dogs, monkeys, cats, and birds) through proper inheritance hierarchies. Most significantly, I implemented a generic AnimalBinaryTree class with recursive insertion, searching, and deletion operations, along with various tree traversal methods (in-order, pre-order, post-order). I created an AnimalTreeManager class that maintains separate binary search trees for each animal type, providing efficient search operations with O(log n) average-case complexity compared to the original O(n) linear searches. The implementation includes advanced search capabilities by multiple criteria including training status, country (both acquisition and service), and age ranges, plus tree performance analytics including height calculation and complexity analysis. I added methods to retrieve available animals (in service and not reserved) efficiently across all trees. This artifact showcases my algorithmic thinking, understanding of computational complexity and Big O notation, ability to implement recursive algorithms, and skill in optimizing system performance through appropriate data structure selection while maintaining clean, maintainable code.

Artifact Three: Database Integration and Security Enhancement
The third artifact demonstrates my database management and security expertise by integrating MySQL database functionality into the Medical Appointment Application from Artifact One. The original Enhancement One version still used in-memory HashMap storage, meaning all data was lost when the application restarted. For Enhancement Three, I designed and implemented a normalized relational database schema with two main tables: patients (storing patient information) and appointments (storing appointment records with patient relationships). I integrated Spring Data JPA and Hibernate ORM to handle object-relational mapping, allowing seamless interaction between Java objects and database tables through JPA annotations (@Entity, @Table, @Column, @Id, @GeneratedValue). The implementation includes a Repository layer using Spring Data JPA's repository pattern, which automatically generates CRUD operations and custom query methods. I added comprehensive transaction management using @Transactional annotations to ensure ACID compliance and data integrity across operations. The enhancement includes automatic audit timestamps (created_at, updated_at) on all records, providing a complete audit trail for compliance requirements. I implemented critical security features including SQL injection prevention through JPA's parameterized queries (prepared statements), input sanitization using regex pattern validation, and prevention of malicious input through Bean Validation constraints. The database integration includes proper indexing on primary keys and unique constraints on business identifiers (patient_code, appointment_code) for query performance optimization. Connection pooling is managed automatically by Spring Boot's HikariCP for efficient resource utilization. This artifact showcases my ability to design normalized database schemas, implement ORM frameworks, write secure database interactions, manage transactions, and create production-ready persistent storage systems that maintain data integrity while protecting against common security vulnerabilities.
